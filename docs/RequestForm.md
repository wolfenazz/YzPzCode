## 11. Description
YzPzCode is a cross-platform desktop application that provides a unified interface for managing and running multiple AI coding agent CLIs including Claude, Codex, Gemini, Opencode, and Cursor. Built with Tauri v2, React, and Rust, it features a terminal grid layout for running multiple AI agent sessions side-by-side, automatic CLI detection and installation, workspace management with custom configurations, and support for Windows, macOS, and Linux.
